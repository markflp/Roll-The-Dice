# 🎲 Dice Game (Ζάρια)

Αυτό είναι ένα παιχνίδι με ζάρια στημμενο με **HTML, CSS και JavaScript**.  
Ο παίκτης ρίχνει δύο ζάρια και αν το άθροισμα είναι **ζυγός αριθμός κερδίζει**, αλλιώς **χάνει**.


---

##  Πώς παίζεται

1. Πατάς το κουμπί **Roll**
2. Ρίχνονται δύο ζάρια
3. Υπολογίζεται το άθροισμα
4. Αν το άθροισμα είναι:
   -  Ζυγό → Κερδίζεις
   -  Μονό → Χάνεις
5. Εμφανίζεται το αποτέλεσμα στην οθόνη

---

##  Τι χρησιμοποιήθηκε

- **HTML** → δομή της σελίδας  
- **CSS** → εμφάνιση και στυλ  
- **JavaScript** → λειτουργικοτητα του παιχνιδιού 

---
 Για να το τρέξεις εγκατάστησε όλα τα αρχεία στον ίδιο  φάκελο και έπειτα τρέξε το αρχείο **index.html**
